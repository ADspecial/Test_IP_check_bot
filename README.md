# Проект Telegram-бота для администрирования сети

## Команды бота 
- `/start` — запуск бота и приветственное слово
- `/help` — справка
- `/check` — подробная проверка ip-адреса
- `/list` — проверка списка адресов и url
- `/file` — проверка адерсов и url из txt файла
